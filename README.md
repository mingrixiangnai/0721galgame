# AstrBot galgame搜索插件

## 简介

在群里输入【/查gal 游戏名称】即可查询内置galgame网站相对应的资源名称和链接

## 配置

本插件无需任何配置。

## 注意事项

*   本插件需要 AstrBot 机器人环境。
*   请确保 AstrBot 机器人已正确安装并配置。
*	搜索不到的资源可能是机器人有时无法解析http到网站

## 示例

**用户：** /查gal 千恋万花

**机器人：** 

	🔍 找到 1 条结果：

	📌 标题：【PC+KRKR】千恋万花
	🔗 链接：https://nn0721.icu/184.html
